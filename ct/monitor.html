  • vvv -> verbose lvl 3
  • --risk 3 -> max risk lvl
  • --level 5 -> max level
  • --technique=T : use only time-based blinds, because the proof-of-concept payload used a timebased blind
  • -p wp_statistics_hit[search_query] in the proof-of-concept payload, this parameter is injected, so might as well try it
